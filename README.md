# 🎧 SosMula Disc Pack

Ресурс-пак для Minecraft, заменяющий все стандартные музыкальные пластинки на треки **SosMula**.

---

## ✨ Особенности

* 🎵 Полная замена всех 22 музыкальных пластинок
* 🔊 Две версии аудио:

  * **SS** — с пространственным звуком (ванильное поведение Minecraft)
  * **WSS** — без затухания звука (фиксированная громкость)
* ⚙️ Оптимизированные OGG-файлы (44.1 kHz, Vorbis)
* 📦 Простая установка без модов

## 📥 Установка

1. Скачайте последнюю версию `.zip`
2. Переместите в папку `resourcepacks`
3. Включите в Minecraft
4. Наслаждайтесь
